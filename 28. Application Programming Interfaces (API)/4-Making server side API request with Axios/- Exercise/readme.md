## 🚀 How to Run Locally

### 1. 📥 Clone the Repository

```bash
git clone https://github.com/yourusername/project-name.git
cd project-name
```

### 2. 📦 Install Dependencies

```bash
npm install
```

### 3. 🏃 Run the App

```bash
node index.js
```

and run you browser and type

```bash
http://localhost:3000/
```
